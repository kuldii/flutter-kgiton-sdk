# Security Policy

## Proprietary Software Notice

This software is the proprietary property of PT KGiTON and contains confidential and trade secret information.

## Reporting Security Vulnerabilities

If you are an **authorized user** and discover a security vulnerability:

### DO NOT:
- ❌ Open a public GitHub issue
- ❌ Discuss the vulnerability publicly
- ❌ Share details with unauthorized parties

### DO:
1. ✅ Email security details to: **security@kgiton.com**
2. ✅ Include your license ID in the subject line
3. ✅ Provide detailed information about the vulnerability
4. ✅ Wait for acknowledgment from PT KGiTON security team

## Security Response

- **Response Time**: Within 48 hours for acknowledged security issues
- **Fix Timeline**: Critical issues will be addressed with high priority
- **Disclosure**: Security patches will be distributed to authorized users only

## Confidentiality

All security-related communications are considered confidential. By reporting a security issue, you agree to:
- Not disclose the issue publicly until PT KGiTON releases a fix
- Not share information with unauthorized third parties
- Follow responsible disclosure practices
